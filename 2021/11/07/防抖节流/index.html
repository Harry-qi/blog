防抖节流

2021-11-07- 2022-07-14

看过很多关于防抖节流的文章，总是记不住原理，最近看视频，发现视频中提供的思路很容易理解

// 防抖
// 思路：fn必须由定时器调用，用户触发就重置定时器
function debounce(fn, delay){
    let timer = null
    return (...args)=>{
        // 用户调用就重置定时器
        if(timer){
            clearTimeout(timer)
        }
        // fn只能由定时器调用
        timer = setTimeout(()=>{
            fn.apply(this, args)
        },delay)
    }
}
// 节流
// 思路：如果当前时间和上次调用时间超过delay就执行fn
