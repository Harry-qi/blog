# 记录下在部署项目时候用到的命令
>  使用linux时间不长，特意记录下平时用到的命令

|  命令   | 含义  |
|  ----  | ----  |
|   ps -ef | grep nginx |  查看进程nginx   |
|  kill -9 <PID>       |  彻底杀死进程   |
|  chmod +x /usr/local/src/jenkins/jenkins.sh   |  设置可执行权限  |
|  cp -r    |    递归复制  |
|  rm -rf   |    递归强制删除文件夹  |
|  cat      |    查看文件  |
|  whereis  |    用于程序名的搜索  |
|  :q!      |    不保存并且强制退出  |
|  :wq      |    保存后离开 vi  |
| tail -f   |    查看日志      |
| top       |    查看cpu占用    |
